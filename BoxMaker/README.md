# BoxMaker-C

**Box Maker** – Kullanıcının girdiği yükseklik ve genişlik değerlerine göre konsolda kutu çizen interaktif bir C programı.

## Kullanım

1. Dosyayı derleyin:  
```bash
gcc boxMaker.c -o boxMaker // Kullandığınız derleyiciye göre
```
2. Uygulamayı çalıştırın.
3. Herhangi iki sayı deneyin.

Dikkat: Konsol boyutunu aşmamaya özen göstermenizi tavsiye ederim. Yoksa kutu şekline ulaşamazsınız.
