# Fibonacci-C

**Fibonacci Dizisi Uygulaması** – Kullanıcıdan alınan sayıya göre Fibonacci dizisini ekrana yazdıran interaktif C programı.

## Kullanım

1. Dosyayı derleyin:  
   ```bash
   gcc fibonacci.c -o fibonacci //Kullandığınız derleyiciye göre
   ```
2. Uygulamayı çalıştırın.
3. Herhangi bir sayı deneyin.

Dikkat: Projede long long kullandım, overflow sorunları yaşayacağınızı sanmıyorum ama bir kısıt olduğunu bilin.
