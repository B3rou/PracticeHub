# isPrime

**Asal Sayı Testi** – Kullanıcıdan alınan sayının asal olup olmadığını kontrol eden interaktif C programı.

## Kullanım

1. Dosyayı derleyin:
```bash
Örnek: gcc isPrime.c -o isPrime -lm // Kullandığınız derleyiciye göre
```
2. Uygulamayı çalıştırın.
3. Herhangi bir sayı deneyin.

Dikkat: Projede Double kullandım, overflow sorunları yaşayacağınızı sanmıyorum ama bir kısıt olduğunu bilin.
