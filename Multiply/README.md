# MultiplicationTablesC

**Multiplication Tables in C** – Kullanıcının girdiği sayıya kadar olan çarpım tablolarını gösteren iki farklı C programı.  
- Birinci program: Tabloları alt alta, her çarpım ayrı satırda olacak şekilde yazdırır.
- İkinci program: Tabloları yan yana ve düzgün bir formatla yazdırır.  

## Kullanım

1. Dosyaları derleyin.
```bash
   gcc multiplyV1.c -o multiply //Kullandığınız derleyiciye göre
   ```
2. Uygulamayı çalıştırın.
3. Kullanıcıdan bir sayı girilmesi istenecek.
4. Programlar girilen sayıya kadar olan çarpım tablolarını ilgili formatta ekrana yazdıracaktır.
