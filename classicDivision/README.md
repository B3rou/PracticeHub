# Bölüm ve Kalan Hesaplama

Bu program, kullanıcıdan iki sayı alır: bölünen ve bölen.  
Program daha sonra bu iki sayıyı kullanarak **bölümü** ve **kalanı** hesaplar ve ekrana yazdırır.

## Kullanım

1. Dosyayı Derleyin.
```bash
gcc classicDivision.c -o classicDivision // Kullandığınız derleyiciye göre
```
2. İlk olarak **bölünen** sayıyı girin.
3. Ardından **bölen** sayıyı girin.
4. Program sonucu ekrana yazdıracaktır.
