# SwapWithTemp

**Swap With Temp** – Kullanıcının girdiği iki sayının değerlerini geçici bir değişken (temp) kullanarak yer değiştiren basit bir C programı.

## Kullanım

1. Dosyayı derleyin.
```bash
gcc swapWithTemp.c -o swapWithTemp // Kullandığınız derleyiciye göre
```
2. Uygulamayı çalıştırın.
3. İki farklı sayı girin.
4. Program sayıların yer değiştirmiş halini ekrana yazdıracaktır.

Dikkat: Girilen değerlerin geçerli sayısal değerler olduğundan emin olun.
