# GradeCalculatorPython

**Grade Calculator** – Kullanıcının girdiği vize ve final notlarına göre ortalamayı ve harf notunu hesaplayan basit bir Python programı.

## Kullanım

1. Dosyayı çalıştırın.
2. Kullanıcıdan vize ve final notlarını girin.
3. Program ortalamayı hesaplar ve karşılık gelen harf notunu ekrana yazdırır.

Dikkat: Girilen değerlerin 0-100 aralığında olduğundan emin olun.
